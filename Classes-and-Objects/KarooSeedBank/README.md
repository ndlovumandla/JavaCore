# Karoo Seed Bank

## Storyline
Deep in the Karoo desert, botanist Dr. Fatima Hendricks manages a seed preservation vault for 400 indigenous South African plant species threatened by climate change. Each seed batch has a species name, collection date, viability percentage, GPS origin coordinates, and a storage temperature requirement. She has been tracking everything in a paper binder for eleven years. A university intern named Sipho is asked to write a Java program to replace the binder — modelling each seed batch as an object, with methods to check viability, flag endangered species, and print storage reports — before the next annual audit.

## How to Run the Project

```bash
# Clone the repository
git clone https://github.com/ndlovumandla/KarooSeedBank.git
cd KarooSeedBank

# Compile the Java file
javac KarooSeedBank.java

# Run the program
java KarooSeedBank
```
